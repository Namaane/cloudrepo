
# Foobar

top_n is a Python library for fidning the top numbers in a array.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install top_n.

```bash
pip install top_n
```

## Usage

```python
import top_n

# returns 'array'
top_n(items,n)
```
